Requirement Analyze#

….It should be able to run commands with arguments, handle input and output redirection, and set up two-element pipelines….Please do not use a memory allocator such as malloc(). Instead you should use only local(stack-allocated) variables and global variables….You don’t need to implement features that aren’t required by the tests, you don’t have to implement parentheses or quoting….

截取了比较关键的信息，从此可以得知：

  1. Shell需要实现的只有指令运行、输入输出的重定向以及双端管道；
  2. 不允许使用动态内存分配；
  3. 测试用例在文件user/testsh.c，Shell只需要通过所有测试用例即可；

分析user/testsh.c，一共有9个测试用例，测试分类为：

  1. test a command with arguments.
  2. test a command, then a newline, then another command.
  3. test output redirection: echo xxx > file
  4. test input redirection: cat < file
  5. test a command with both input and output redirection.(grep pointers < README > testsh.out)
  6. test a pipe with cat filename | cat
  7. test a pipe that has both redirection and a pipe.
  8. ask the shell to execute many commands, to check if it leaks file descriptor.

重点注意5、7、8条。
